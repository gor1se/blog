Creating a simple login Form

In this first excersice we will learn how to create a simple but well designed login form.

Step 1: Creating all nesseccary files

In this first steps we are going to create all the nessessary files for our project. If you feel comfortable enough, you can do so in the command shell. We are going to create an index.html and a styles.css file.

cd dev // Go to the directory where you'd like to start this project in
mkdir loginForm // Creating the project folder
cd loginForm // Changing into our newly created project
touch index.html styles.css // Creating the nesseccary files with "touch"

Step 2: HTML

Whenever you open up a brand new HTML file with the intention of creating a new weppage, it will always be a good idea to start with a boiler-plate HTML structure like this. You can just copy and paste this code to get startet or use the Emmet-Abreviation "!" and then press "tab".

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
</body>
</html>
